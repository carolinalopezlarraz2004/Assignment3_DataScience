# Assignment3_DataScience

## Files

- **generating_file.py**  
  Script used to generate the 100 JSON provenance records (dataset simulation).

- **Third_Assignment.R**  
  Main R script containing the Shiny dashboard and all data processing.

- **Assignment3_Report.pdf**  
  Report explaining the project, methodology, and results.

- **README.md**  
  Project description.

## Description

This project implements a Shiny dashboard to monitor pipeline executions, including performance metrics (duration, failures, data size) and provenance information (agents, entities, relationships).
